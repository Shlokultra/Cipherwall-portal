# CipherWall - Cybercrime Reporting Portal

## Overview
CipherWall is a full-stack cybercrime reporting and awareness portal designed to allow users to submit incidents securely and access cybersecurity resources.

## Tech Stack
- HTML5
- CSS3
- Bootstrap
- PHP
- MySQL
- PHPMailer

## Key Features
- Secure user authentication
- Incident reporting system
- Admin dashboard for report management
- Email notifications
- Structured backend architecture

## Project Structure
- /admin
- /assets
- /config
- index.php

## Setup Instructions
1. Clone the repository
2. Import database into MySQL
3. Configure dbconnect.php
4. Run via XAMPP
